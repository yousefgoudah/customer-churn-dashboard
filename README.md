# 🚀 Customer Churn Intelligence Platform

## 🎯 Business Problem

Telecom companies lose revenue when customers cancel services.  
This project predicts churn risk early so retention teams can act before losing customers.

## 🧠 Solution

- Predict churn probability in real time
- Segment customers by risk
- Generate retention recommendations
- Bulk score CSV customer lists

## 🛠️ Tech Stack

- Python
- Pandas
- Scikit-learn
- Random Forest
- Streamlit
- FastAPI
- Plotly

## 📈 Model Performance

- Accuracy: 73.8%
- ROC-AUC: 0.835
